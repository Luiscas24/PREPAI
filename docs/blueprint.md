# **App Name**: PrEPPal

## Core Features:

- Dashboard Status: Display current PrEP status (dose taken, pending, missed) with color-coded indicators.
- Dose Logging: Allow users to quickly log their daily PrEP dose with a floating action button.
- Supply Tracking: Track remaining PrEP supply, displaying days and pill count with low-supply alerts.
- Adherence Monitoring: Monitor PrEP adherence via a monthly calendar with color-coded dose tracking (taken, missed, not registered).
- Appointment Scheduling: List upcoming medical appointments and medication delivery visits with configurable reminders.
- Refill Management: Manage prescription refills and track the administrative process.
- Personalized Recommendations for Timing Dose: An AI powered tool which analyses the user's location to send push notifications reminding them to take the medication, to assist with adherence. 

## Style Guidelines:

- Primary color: Saturated blue (#29ABE2) to represent health, trust, and technology. 
- Background color: Light, desaturated blue (#E0F7FA) for a clean and calming interface.
- Accent color: Green (#90EE90) to highlight positive actions and confirmations, providing visual reinforcement for adherence.
- Body and headline font: 'Inter' (sans-serif) for a modern and readable design.
- Code font: 'Source Code Pro' for use when displaying discrete reminders such as notification text set by the user.
- Use clear, intuitive icons to represent medication, appointments, and reminders.
- Implement a clean, card-based layout with clear calls to action for primary functions like logging doses and scheduling appointments.
- Use subtle transitions and animations to provide feedback on user actions, such as dose logging or appointment confirmations.
